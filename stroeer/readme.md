# tAPIr

_tAPIr_ is acronym for **t**-online **API** **r**epository. It consists of the following components:

1. `core` - the core models and services. It is the foundation of the whole project.
2. `page` - the page model and services. This is mainly used for the page rendering, 
and therefore it contains quite project specific code. 
3. `curation` - the curation model and services. This is mainly used for the 
editorial curation of our content, e.g. lists of articles.
4. `fragment` - contains a fragment view for the page rendering by external
partners and entities.
5. `navigation`- exposes the site navigation as a service.
